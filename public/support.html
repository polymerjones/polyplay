What PolyPlay Audio Does

PolyPlay Audio lets you build and play a personal media library from your own files.

  • Import audio from supported local files.
  • Organize tracks into playlists.
  • Replace artwork, including video artwork on supported flows.
  • Use offline playback, loops, backups, and custom visual themes.

Compatibility Notes

PolyPlay Audio is designed around local-device playback and local storage.

  • Behavior can vary slightly between desktop browsers, Safari, and the iPhone app wrapper.
  • Large media files, unsupported codecs, or browser storage limits can affect import behavior.
  • Video artwork support depends on the browser or platform decoding the uploaded file correctly.

Troubleshooting

  • If imported media does not appear correctly, try re-importing the file from the app’s import/settings flow.
  • If playback or artwork looks stale, fully close and reopen the app, then revisit the affected track.
  • If a backup import/export seems incomplete, confirm that the file finished saving before re-importing it.
  • If local media was removed from the device or browser storage was cleared, the app may no longer be able to load that content.

Recent release focus

  • Loop editor: refined handles stay centered without zooming out, fullscreen exit is locked during edits, and desktop selection is suppressed while dragging.
  • Desktop player: double-click toggles and selective title copying improve the polish without affecting controls.
  • We are actively watching playback/resume paths so cropped loops never briefly start from the beginning.

Backups And Data Safety

PolyPlay Audio is intended to store your library data locally. If your media, playlists, settings, or gratitude entries matter to you, create backups regularly.

  • Use the app’s backup/export tools before deleting the app or clearing browser/device storage.
  • Deleting local app data can permanently remove your library unless you saved a backup first.
  • Backups should be stored somewhere you can find later, such as Files, iCloud Drive, or your preferred desktop folder.

What To Include In A Support Message

  • Your device and OS version.
  • Browser name/version or whether you are using the iPhone app.
  • What action you took right before the issue.
  • What you expected to happen.
  • What actually happened.
