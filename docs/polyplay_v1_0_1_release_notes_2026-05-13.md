# PolyPlay Audio v1.0.1 Release Notes

Release type: polish and bug-fix update.

## App Store Notes

- Added App Store, Instagram, and TikTok links to Settings.
- Improved newly imported track playback so a new track can start from the first tile or row tap.
- Improved artwork reliability when switching between tile and row views.
- Polished Settings/Admin transition visuals.
- Added minor UI polish across the main player and Settings surfaces.

## Build Values

- Marketing/App Store version: 1.0.1
- iOS build number: 2

## QA Checklist

- Import a fresh track and confirm first tap playback on device.
- Switch tile/list views with uploaded artwork and confirm artwork remains visible.
- Open Settings/Admin repeatedly on iPhone and confirm no logo flash/regression.
- Confirm Settings footer links open externally.
- Confirm main toolbar layout on iPhone and desktop widths.
